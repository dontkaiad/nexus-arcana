**ENV:** `NOTION_DB_ERRORS`
**ID:** `31a42b3b1ac080558c68fe885ece5b2a`

| Поле         | Тип    | Значения                                                                 |
|--------------|--------|--------------------------------------------------------------------------|
| Сообщение    | Title  | текст сообщения пользователя                                             |
| Дата         | Date   | YYYY-MM-DD                                                               |
| Тип ошибки   | Select | `processing_error` · `api_error` · `notion_error` · `parse_error` · `unknown_type` |
| Код          | Select | `529` · `5xx` · `4xx` · `timeout` · `ParseError` · `–`                  |
| Бот          | Select | `☀️ Nexus` · `🌒 Arcana`                                                 |
| Ответ Claude | Text   | raw ответ Claude для отладки промптов                                    |
| Трейсбек     | Text   | Python traceback                                                         |

**Как читать:**
- `api_error` + `529/5xx` → серверы Anthropic, ждать
- `notion_error` + `5xx` → Notion упал, ждать
- `processing_error` + `–` → наш код сломан, Кай правит
- `parse_error` → промпт неправильный, Кай правит
- `unknown_type` → юзер написал что-то непонятное, норм
- `api_error` + `4xx` → протух токен, Кай правит
